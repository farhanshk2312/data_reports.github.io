Credit Risk Analysis

project summary


This project focuses on analyzing large-scale loan datasets to uncover patterns in borrower behavior, default risk, and repayment efficiency. Using BigQuery for raw data storage and Databricks for advanced processing, the workflow covers data cleaning, feature engineering, and deep exploratory data analysis.

The goal is to build insights and features that support credit risk modeling and predictive analytics, ultimately helping financial institutions make more data-driven lending decisions.

Data Overview


The dataset includes detailed borrower and loan-level information such as loan amount, funded amount, interest rate, term, issue date, borrower demographics, employment details, debt-to-income ratios, revolving balances, delinquency history, and recovery outcomes. The data was cleaned, parsed, and enriched with engineered features like repayment behavior, utilization ratios, and financial stress indicators.

Data Source


The data is derived from publicly available loan performance datasets widely used for research and modeling in credit risk analytics.

This dataset contains complete loan data for all loans issued through the year 2007 - 2015, including the current loan status (Current, Late, Fully Paid, etc.) and latest payment information. It contains total of 8,55,969 records with 73 features including target variable.

The raw data was first ingested into Google BigQuery, then loaded into a Databricks catalog for transformation and advanced analysis.

key Objectives

1. Understand loan distribution and borrower demographics.

2. Explore correlations between loan characteristics and default risk.

3. Identify patterns in credit behavior, repayment, and financial health.

4. Provide recommendations for lending strategies and risk mitigation.
